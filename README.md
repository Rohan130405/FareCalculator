Fare Calculator 

This is a simple Python application that calculates travel fare based on distance, vehicle type, and time.

The user enters the distance, selects a vehicle type such as Economy, Premium, or SUV, and provides the time of travel. The system calculates the fare based on predefined rates. Additional charges are applied during peak hours.

This project focuses on basic programming concepts such as conditional statements, functions, and user input handling. It also includes input validation to handle incorrect values.

The project helped me improve my understanding of logic building and writing clean, user-friendly programs.
